# 2D宇宙太空战物理模拟 - GUI界面

基于pygame开发的图形用户界面，提供完整的2D宇宙物理模拟控制和可视化。

## 功能特性

### 核心功能
- **实时物理模拟**：基于牛顿力学和万有引力定律的2D宇宙模拟
- **多对象支持**：支持恒星、行星、小行星等实体和飞船等对象
- **观测延迟**：考虑光速延迟的真实观测系统
- **轨道预测**：实时计算和显示对象轨道

### 用户界面
- **1920×1080全高清窗口**：优化的视觉体验
- **双界面模式**：观测界面和机动界面
- **信息面板**：实时显示对象状态和模拟信息
- **轨道可视化**：彩色轨道线显示

### 控制功能
- **相机控制**：WASD移动视野，QE缩放
- **时间控制**：,.调整时间倍速，Space暂停/继续
- **参考系切换**：G键切换不同参考系视角
- **对象选择**：Tab键循环选择对象
- **界面切换**：T键切换观测/机动界面

### 机动系统
- **鼠标控制**：点击选择机动方向
- **参数调整**：O/-调整总dv，P/[调整角度，]/[调整节流
- **机动执行**：Enter确认执行机动

## 快速开始

### 运行要求
- Python 3.7+
- Pygame 2.6.1+
- NumPy

### 安装依赖
```bash
pip install pygame numpy
```

### 启动程序
```bash
python gui_main.py
```

## 配置文件

程序使用`config.json`配置文件，包含以下主要设置：

### 窗口设置
- 分辨率：1920×1080
- 全屏模式：可配置
- 标题：2D宇宙太空战物理模拟

### 模拟参数
- 时间步长：60秒
- 最大时间倍速：1000倍
- 模拟范围：1万亿米
- 引力常数：6.67430e-11

### 初始场景
默认包含太阳系演示场景：
- 太阳：中心恒星
- 地球：轨道行星
- 火星：轨道行星
- 探索者号：可操控飞船

### 颜色方案
- 背景：深蓝色太空
- 网格：浅灰色参考线
- 实体：按类型着色（恒星黄色，行星蓝色等）
- 轨道：绿色轨迹线

## 控制说明

### 观测界面控制
```
WASD      - 移动相机视野
Q/E       - 缩放视野
,/.       - 减少/增加时间倍速
Space     - 暂停/继续模拟
G         - 切换参考系
T         - 切换到机动界面
Tab       - 切换选中对象
```

### 机动界面控制
```
鼠标移动  - 选择机动方向
O/-       - 增加/减少总dv
P/[       - 增加/减少角度
]/[       - 增加/减少节流深度
Enter     - 确认执行机动
T         - 返回观测界面
Space     - 暂停/继续模拟
```

## 系统架构

### 主要模块
1. **gui_manager.py** - GUI主控制器
   - 处理用户输入
   - 管理渲染循环
   - 协调模拟和界面

2. **physics/simulation.py** - 物理模拟引擎
   - 整合重力、运动、碰撞模块
   - 管理对象和实体
   - 提供模拟API

3. **time_manager/time_manager.py** - 时间管理系统
   - 管理模拟时间
   - 处理参考系切换
   - 控制时间倍速

4. **observation/observer.py** - 观测系统
   - 处理光速延迟
   - 坐标转换
   - 可见性计算

### 数据流
```
用户输入 → GUI管理器 → 模拟引擎 → 物理计算
    ↑           ↓           ↓           ↓
界面渲染 ← 状态更新 ← 观测系统 ← 时间管理
```

## 扩展开发

### 添加新实体类型
1. 在`config.json`的`initial_scenario.entities`中添加新实体定义
2. 在`gui_manager.py`的`_load_initial_scenario`方法中添加创建逻辑
3. 在`draw_entities`方法中添加渲染支持

### 修改控制方案
1. 在`config.json`的`controls`部分修改键位映射
2. 在`gui_manager.py`的`handle_input`方法中更新处理逻辑

### 自定义场景
编辑`config.json`中的`initial_scenario`部分：
- 修改现有实体参数
- 添加新实体和对象
- 调整初始位置和速度

## 故障排除

### 常见问题
1. **窗口无法打开**
   - 检查pygame安装：`pip install pygame`
   - 确认Python版本兼容性

2. **模拟运行缓慢**
   - 减少时间倍速
   - 关闭轨道预测显示
   - 减少实体数量

3. **控制无响应**
   - 确认窗口焦点
   - 检查键盘布局
   - 验证配置文件格式

4. **显示异常**
   - 检查显卡驱动
   - 调整窗口分辨率
   - 验证颜色配置

### 调试模式
程序启动时会显示控制说明和加载状态，关注控制台输出获取调试信息。

## 性能优化建议

1. **渲染优化**
   - 根据缩放级别动态调整网格密度
   - 实体过小时不显示标签
   - 使用简单的几何图形表示对象

2. **模拟优化**
   - 调整时间步长平衡精度和性能
   - 限制最大对象数量
   - 使用空间分区优化碰撞检测

3. **内存管理**
   - 限制历史记录大小
   - 定期清理不再需要的参考系
   - 使用numpy数组优化数值计算
